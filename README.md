# 春日影 (MyGO!!!!! Ver.)

使用[AMLL](https://github.com/Steve-xmh/applemusic-like-lyrics)制作。

![](https://i.postimg.cc/sD3yYF91/image.png)

桌面端浏览器用 `<Shift-D>` 快捷键切换 Debug 模式：

![](https://i.postimg.cc/63Vgq3Pc/image.png)

按：移动端浏览器可以在右下角连续点击 5 次来切换 Debug 模式。
